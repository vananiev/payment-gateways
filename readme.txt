Allows you to use Bank Card, Yandex.Money, Webmoney, Robokassa, Qiwi payment gateway with the WooCommerce plugin.

== Description ==

После активации плагина через панель управления в WooCommerce прописывем
AppId и Secret узнать их можно в [личном кабинете](http://russian-payments.ru)


В личном кабинете (http://russian-payments.ru) прописываем:
<ul style="list-style:none;">
<li>Result URL: http://your_domain/?wc-api=wc_rpg&about=result</li>
<li>Success URL: http://your_domain/?wc-api=wc_rpg&about=success</li>
<li>Fail URL: http://your_domain/?wc-api=wc_rpg&about=fail</li>
</ul>

Если возникнут проблемы, [сообщите об ошибке](mailto:support@russian-payments.ru)


== Installation ==
1. Убедитесь что у вас установлена посленяя версия плагина [WooCommerce](/www.woothemes.com/woocommerce)
2. Распакуйте архив и загрузите содержимое в папку ваш-домен/wp-content/plugins
3. Активируйте плагин
